# Inputs

There are a variety of inputs available in the library. The inputs are designed to provide type safe inputs into forms.

- TextInput is the basic input and don't include any validations
- DateInput uses the HTML5 date types, there is a polyfill that is needed for support of Safari and IE < 11
- NumericInput uses the HTML5 numeric type
- SelectInputh offers a standard option

Since the library tries to use HTML5 standards wheneever possible the implementation may differ slightly across
browsers.

# DataTable

A DataTable is an easier way to visualize arrays of data. You can add an array of data to define the column type and
the properties of the column.

DataTables will expand to fill the area of surroding container.

You can also add a function to format the data so the array of initial data should remain unformatted to enable proper
sorting, while presenting appropriate data to users.

export const bold = (string) => (
  <span style={{ fontWeight: "bold" }}>{string}</span>
);

export const Template = (args) => (
  <DataTable
    columns={[
      {
        header: "Strings",
        textAlign: "left",
        formatFunction: bold,
      },
      {
        header: "Numbers",
        textAlign: "center",
        width: "10rem",
        sortable: true,
      },
      {
        header: "Dates",
        textAlign: "right",
        width: "25rem",
      },
    ]}
    data={[
      ["a", 1, Date(2021, 0, 1)],
      ["b", 2, Date(2021, 1, 1)],
      ["c", 3, Date(2021, 2, 1)],
    ]}
    {...args}
  />
);
